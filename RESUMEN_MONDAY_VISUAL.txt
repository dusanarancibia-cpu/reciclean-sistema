┌────────────────────────────────────────────────────────────────────────────────┐
│  ⚙️ AUTOMATIZACIONES (6)                                                      │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  1. "Tarea vence HOY"                                                         │
│     └─ Trigger: Fecha Límite = TODAY                                          │
│     └─ Notificación: Responsable + Dusan a las 8 AM                          │
│     └─ Mensaje: "⏰ {Tarea} VENCE HOY. Siguiente: {Acción}"                 │
│                                                                                │
│  2. "Tarea vence en 2 días"                                                   │
│     └─ Trigger: Fecha Límite = TODAY + 2                                      │
│     └─ Notificación: Responsable a las 4 PM (amarillo)                       │
│     └─ Mensaje: "⚠️ {Tarea} vence en 2 días"                                 │
│                                                                                │
│  3. "Bloqueada > 3 días"                                                      │
│     └─ Trigger: Bloqueador NOT EMPTY + (Hoy - Última actualización) > 3d     │
│     └─ Notificación: Responsable + responsable del bloqueador                │
│     └─ Mensaje: "🔴 {Tarea} bloqueada desde {fecha}"                        │
│                                                                                │
│  4. "Sub-tarea completada"                                                    │
│     └─ Trigger: Sub-tarea % = 100%                                            │
│     └─ Action: Actualizar % padre automático                                  │
│     └─ Notificación: Equipo                                                   │
│                                                                                │
│  5. "Idea aprobada sin asignar"                                               │
│     └─ Trigger: Tablero 3, Estado = Aprobada + (Ahora - cambio) > 1d        │
│     └─ Notificación: Dusan a las 10 AM                                        │
│     └─ Mensaje: "👍 Idea '{idea}' aprobada. ¿A quién la asigno?"           │
│                                                                                │
│  6. "Resumen diario" ⭐                                                       │
│     └─ Trigger: Lunes-viernes a las 8 AM + 5:30 PM                          │
│     └─ Action: Post a Slack o email                                           │
│     └─ Contenido:                                                             │
│        ├─ 8 AM: "Tareas de hoy (X personas, Y tareas)"                       │
│        │        - Desglose por persona                                        │
│        │        - Tareas vencidas                                             │
│        │        - Bloqueadores activos                                        │
│        │        - Link a vista "Hoy"                                          │
│        └─ 5:30 PM: "Cierre de día:"                                          │
│                    - Completadas (verde)                                      │
│                    - En curso (amarillo)                                      │
│                    - Bloqueadas (rojo)                                        │
│                    - % avance diario                                          │
│                                                                                │
