## Como leer cada ficha

Cada archivo `ficha-*.md` contiene **8 tablas** estandarizadas:

| Seccion | Tema |
|---------|------|
| A | Identidad (empresa, pais, tamano) |
| B | Que hacen (productos y propuesta valor) |
| C | Como lo hacen (flujo operativo paso a paso) |
| D | Herramientas / tecnologias que ocupan |
| E | Como se ve la informacion (UI, dashboards, apps) |
| F | Que tenemos nosotros vs ellos |
| G | Brecha e implementacion (dificultad + esfuerzo) |
| H | Fuentes consultadas |

1. **Lee primero** `TABLA-EJECUTIVA.md` - tiene todo resumido en 6 tablas.
